# Image Converter

## Overview
This is a simple Python script that converts images from one format to another. It can handle a single file or an entire folder of images.

---

## Features
- Convert between common formats (jpg, png, webp, etc.)
- Works on a single image or a whole directory
- Automatically creates an output folder if it doesn’t exist

---

## Requirements
- Pillow library

Install Pillow with:
pip install pillow
